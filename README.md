# StreamBop
Stream bop is a OBS utility and allows music to play automatically without you worrying about music.

# Downloading
Windows: Releases
Mac: Not Suppported
Linux: Use Snap
[![Snap](https://snapcraft.io/streambop/badge.svg)](https://snapcraft.io/streambop)

```
snap install streambop
```
